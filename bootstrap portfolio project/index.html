Developer Skills

Below is a list of various languages, libraries, and frameworks I currently feel comfortable with as a frontend developer.

  1. The Basics

    • HTML

      • Uncommonly known as "Hyper Text Markup Language"
      • HTML has defined the basic webpage structure for nearly 35 years, including this one.

    • CSS

      • CSS describes how HTML elements should be displayed.
      • When tags like <font> were first introduced, many developers faced a long a tedious process if they wanted to add them to entire parts of their website.
      • In came CSS, which allowed developers to style entire websites in as little as one line of code.

    • JavaScript

      • "The programming language of the web."
      • Arguably the most crucial tool of the big 3, JS allows developers to provide a more engaging user experience by being able to manipulate entire web structures through various inputs from the user or the browser itself.
      • For Example: The red loading bar at the top of this page updates live using JavaScript.

  2. Additional Tools

    • Bootstrap

      • Bootstrap is a popular open-source framework that provides a collection of pre-built HTML, CSS, and JavaScript components and tools.
      • This webpage utilizes Bootstrap at nearly every step of the way from the navbar to the contact page.
      • Overall, Bootstrap empowers developers to create modern, responsive, and visually appealing websites and web applications with less effort and time.

    • React

      • React is a popular JavaScript library used for building user interfaces in web applications.
      • Overall, React simplifies the process of building interactive and responsive user interfaces, encourages code reusability, and provides efficient rendering, making it a popular choice for web developers.

    • jQuery

      • Vastly simplifies the process of interacting with HTML documents, handling events, performing animations, and making Ajax requests.
      • jQuery offers a wide range of utility functions that facilitate common programming tasks, such as iterating over arrays or objects, manipulating CSS styles, working with animations, and handling browser compatibility issues.
